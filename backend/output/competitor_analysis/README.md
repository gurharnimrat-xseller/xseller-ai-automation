# Competitor Analysis Files

This directory contains analysis of the target competitor video quality.

## Files Included

- `transcript.txt` - Full transcript of competitor video
- `technical_specs.json` - Video technical specifications (resolution, fps, codec, etc.)
- `competitor_video.en.vtt` - Subtitles/captions file
- `competitor_video.mp4` - **NOT included in repo** (13MB)

## Getting the Competitor Video

The competitor video is excluded from the repository due to size (13MB).

**To download it:**

```bash
cd backend/output/competitor_analysis
yt-dlp https://www.youtube.com/shorts/8suzvBwacZI -o competitor_video.mp4
```

Or use the analysis files already included - they contain all the specs needed to replicate the quality!

## Analysis Outputs

These files were generated by Codex's competitor analysis scripts and contain:
- Exact video specifications to match
- Voice characteristics and pacing
- B-roll timing patterns
- Quality benchmarks

Use these as the target for our video generation!
